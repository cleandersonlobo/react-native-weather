 ## Get Started
 
 #### 1. Clone the Repo
 
 On the command prompt run the following commands
 ```sh
 $ git clone https://github.com/cleandersonlobo/react-native-weatcher.git
 
$ cd react-native-weather
 
$ npm install or yarn 

# ios
$ cd ios/ && pod install 

$ react-native run ios or android
 
 ```
